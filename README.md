# Bv2Av
B站Bv号转Av号

Bv Number to Av Number for Bilibili(China)

请在你运行"Bv2Av.py"之前执行命令```pip install pyperclip```。

Please run the command ```pip install pyperclip``` before you run "Bv2Av.py".
